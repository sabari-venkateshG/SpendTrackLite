# **App Name**: SpendTrack Lite

## Core Features:

- Simple Expense Input: Allow users to quickly add expenses with just amount and reason.
- Image Expense Capture: Extract key details (amount, vendor, date) from uploaded images or taken pictures using AI tool, then map to expense fields.
- Expense Categorization: Automatically categorize expenses (e.g., food, transport, utilities).
- Local Storage: Store expense data locally on the user's device.
- Reporting Dashboard: Display monthly, 6-month, and annual expense reports with clear visualizations.

## Style Guidelines:

- Primary color: Calm blue (#64B5F6) to promote trust and ease of use.
- Background color: Light gray (#F0F4F7), almost white, for a clean, uncluttered look.
- Accent color: Soft green (#A5D6A7) to highlight positive actions or summaries.
- Body and headline font: 'PT Sans' for a balance of modernity and approachability.
- Use minimalist icons to represent expense categories and navigation elements.
- Keep a clean and intuitive layout with a focus on easy navigation.
- Use subtle animations to acknowledge user input and transitions.
